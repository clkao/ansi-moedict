### ANSI-Moedict

Draw moedict-style ansi-art in your terminal

### Dependencies

[ansi-canvas](https://github.com/TooTallNate/ansi-canvas)

[commander](https://npmjs.org/package/commander)

### Usage

Help

```
$ node index.js -h
```
Command
```
$ node index.js moedict 拆政府
```
